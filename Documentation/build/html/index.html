wukong-simulator 文档¶

wukong-simulator 是用于开发 wukong-firmware 的模拟器，其中包含了NVMe Controller和Flash Controller的寄存器级别模拟，可以通过连接 fio 作为host进行测试。

注解

This project is under active development.
