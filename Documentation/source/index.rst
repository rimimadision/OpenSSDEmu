wukong-simulator 文档
=====================

`wukong-simulator <https://e.gitee.com/openstorage_1/repos/openstorage_1/wukong-simulator>`_ 
是用于开发
`wukong-firmware <https://e.gitee.com/openstorage_1/repos/openstorage_1/wukong-firmware>`_
的模拟器，其中包含了NVMe Controller和Flash Controller的寄存器级别模拟，可以通过连接
`fio <https://github.com/axboe/fio>`_ 
作为host进行测试。

.. note::

   This project is under active development.
